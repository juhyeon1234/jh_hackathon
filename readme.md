# 🍱 서울숲 구내식당 - AI 스마트 식당 플랫폼

> 구내식당 메뉴 탐색부터 AI 추천, 건강 기록, 메뉴 투표, 사전 예약까지 — 직원의 점심 경험을 AI로 완전히 새롭게 만드는 올인원 플랫폼

---

## 📌 프로젝트 개요

**문제 정의**
직장인에게 점심시간은 하루 중 몇 안 되는 휴식이지만, 매일 반복되는 메뉴 고민과 긴 대기줄, 알레르기 불안, 놓치는 건강 관리로 인해 오히려 스트레스가 된다. 구내식당은 매일 이용하는 공간임에도 불구하고 디지털 경험은 여전히 단순 공지 수준에 머물러 있다.

**솔루션**
AI를 식당 경험의 중심에 놓는다. 메뉴 추천부터 영양 분석, 참여형 소통까지 — 직원 개개인에게 맞춤화된 점심 경험을 제공한다.

---

## 🎯 핵심 기능

| 기능 | 설명 |
|------|------|
| 📅 주간 메뉴 | 주차별 코너별 메뉴 조회 + 오늘 메뉴 하이라이트 + 투표 |
| 🤖 AI 메뉴 추천 | 기분·시간·기피 음식 입력 → Claude API 맞춤 추천 (API 키 없이도 룰 기반 동작) |
| 🗳️ 먹고싶어요 | 원하는 메뉴 제안 및 투표 (API 키 있을 시 AI 유사어 중복 감지) |
| 📋 사전 예약 | 테이크아웃 메뉴 시간대별 사전 예약 및 전체 현황 조회 |
| 💪 건강 기록 | 메뉴 선택 시 자동 영양 분석(AI 또는 기본값) 및 주간 리포트 |
| ⚠️ 알레르기 관리 | 개인 알레르기 설정 시 해당 메뉴에 자동 경고 + 투표 시 토스트 알림 |

---

## 🛠️ 기술 스택

- **Frontend**: Vanilla HTML/CSS/JavaScript (단일 파일, 무의존성)
- **AI**: Anthropic Claude API (`claude-sonnet-4-20250514`)
  - 메뉴 추천, 영양 분석, 중복 감지, 건강 리포트
  - API 키 미입력 시 룰 기반 기본 모드로 자동 대체
- **데이터 저장**: In-memory (서버·DB 없이 동작, 새로고침 시 초기화)
- **배포**: GitHub Pages

### AI 기능 활성화 (선택)
상단 바에 Anthropic API 키(`sk-ant-...`) 입력 시 Claude AI 기능 풀 활성화.
미입력 시에도 룰 기반 추천·기본 영양값·기본 리포트로 앱 전체 정상 동작.

## 📊 차별점

1. **AI-Native 설계** — 단순 메뉴 표시를 넘어 모든 주요 기능에 AI 내재화
2. **Zero-install / Zero-dependency** — 서버·DB·패키지 없이 브라우저만으로 완전 동작
3. **Graceful Degradation** — API 키 없이도 모든 기능이 기본 모드로 동작
4. **개인화** — 알레르기, 칼로리 목표, 기분 기반 완전 개인화 경험
5. **참여형** — 직원이 직접 메뉴를 제안하고 투표하는 쌍방향 구조
